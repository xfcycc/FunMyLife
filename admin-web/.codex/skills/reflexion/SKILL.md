# Reflexion (自我反思)

E 阶段每个 Task 实现完成后, Micro-review 之前。

## 反思清单
1. 走捷径了吗? (as any, 空 catch)
2. 忽略边界了吗? (空/超长/并发/null)
3. 硬编码了吗? (URL/端口/阈值)
4. 过度工程了吗? (需求没要求的功能)
5. 偏离 spec 了吗?
6. 6 个月后能理解吗?

发现问题 → 立即修复
